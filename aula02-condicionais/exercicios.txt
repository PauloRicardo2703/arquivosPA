Exercícios 

1- Desenvolva um algoritmo que receba dois valores e
imprima no console o maior deles. Usar a estrutura 
de decisão "IF" 

2- Desenvolva um algoritmo para imprimir na tela se 
o voto do(a) usuario(a) é obrigatório , sabendo que 
a idade dela(a) é 18anos. Usar  apenas o IF

3- Desenvolva um programa para calcular a media de um (a) estudante com base
em duas notas. Se a nota for maior ou igual a 7, deverá apresentar
a mensagem: "Você está aprovado(a) com a media (inserir a media calculada). 
Caso
contrario , deverá apresentar a mensagem "você está reprovado(a) com a media 
(inserir a media calculada). Você precisa estudar mais!".
